Data Types in Python

A data type defines the kind of value a variable can store and determines what operations can be performed on it.

Example

x = 10
price = 99.5
name = "Python"
is_active = True
  • x is an integer
  • name is a string
  • price is a float
  • is_active is a boolean.

Key Points

  • Python automatically assigns a data type based on the value
  • No need to declare the data type explicitly
  • Data types help Python perform correct operations

Common Built-in Data Types

  • int whole numbers (10, -5, 0)
  • float decimal numbers (3.14, 0.5)
  • str text values ("Python")
  • bool True or False
  • list ordered and mutable collection
  • tuple ordered and immutable collection
  • dictkey-value pairs

Practice Questions

  1. Create variables of type int, float, and str
  2. Create a list with three items
  3. Create a dictionary to store your name and age

FAQs

Q1. Do we need to specify data types in Python?

No. Python automatically assigns a data type based on the value.

Q2. Can a variable change its data type?

Yes. Python supports dynamic typing.

Q3. How can I check the data type of a variable?

By using the type() function.
