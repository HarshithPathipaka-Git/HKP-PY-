Variables in Python

A variable is a name used to store a value in memory so it can be reused later.

Example

            x = 10
            name = "Python"
            is_active = True

Here, x stores a number, name stores text, and is_active stores a boolean value.

Key Points

  • No need to declare a variable type in advance
  • Python decides the type at runtime
  • Variable names should be meaningful

Practice Questions

  1. Create a variable to store your age
  2. Create a variable to store your name and print it
  3. Change a variable value and print the updated value

FAQs

Q1. Can a variable change its value?

Yes, variables in Python can be reassigned to new values.

Q2. Do variables need an explicit data type?

No, Python automatically assigns a data type based on the value.

Q3. What is boolean data?

Boolean values represent True or False
